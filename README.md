## Install

    ```
    npm install
    ```
## Run
- run all tests 
    ```
    npm run test
    ```
- run single test headless
    ```
    npm run test example.spec.ts
    ```
- run single test in headed mode
    ```
    npm run test --headed
    ```
- run chosen file test or files
    ```
    npm run test tests
    ```
